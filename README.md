This project is my first try at **vibe coding** and is being developed for a real business, **Funding Connect Consulting**. Because this website represents a real organisation, the business name, content, branding and related data are not available for reuse.

The website is for **Funding Connect Consulting**, a First Nations-focused consulting business that supports Aboriginal and Torres Strait Islander organisations with funding opportunities, grant writing, program design and community-led impact.

The goal of this project is to create a professional, accessible and easy-to-maintain website that can be launched through **Vercel** and expanded over time with live grant updates and newsletter functionality.

# 📌 Project Overview

Funding Connect Consulting is designed to help connect Aboriginal and Torres Strait Islander organisations with funding that supports meaningful, community-led outcomes.

The website currently includes:

- A professional landing page
- An About section explaining the company’s purpose and values
- A Services section outlining consulting support areas
- A Connect page with enquiry and newsletter forms
- A grants slider for funding opportunities
- Responsive styling for desktop and mobile
- A clean static setup ready for Vercel deployment

The project is currently a static website, with future plans to integrate real newsletter functionality and regular updates on local grant opportunities.
